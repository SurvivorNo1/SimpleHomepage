卡特兰数(catalan number)

定义

卡特兰数是一种出现在组合数学中的数列。

假设我们有一个数列 Cn，C0 = 1，并且 Cn 是前 n 个卡特兰数的线性组合。卡特兰数 Cn 可以由以下的公式计算出：

Cn=1n+1C2nn=2n!(n+1)!n!

这个数列从 n=0 开始，前几个数是：1, 1, 2, 5, 14, 42, 132, ...

卡特兰数在很多数学问题中都有应用，以下是一些例子：

  1. 二叉树：对于 n 个结点的(国际)满二叉树（也称为满二叉搜索树或者满二叉排序树），其可能的形状数量就是第 n 个卡特兰数。(国内国外定义不同)

  2. 凸多边形对角线：对于一个凸的 n+2 边形，可以通过 n-1 条不相交的对角线将其分成 n 个三角形。这样做的方法数量也是第 n 个卡特兰数。

  3. 化学：在化学中，卡特兰数也被用于计算某些特定类型的化学结构的数量，比如全烷烃的异构体数量。

  4. 乒乓球比赛：在一场单淘汰赛（比如乒乓球）中，如果一个选手从第一轮比赛到最后的冠军，总共需要赢 n 场比赛。如果我们用括号来表示比赛的输赢（比如，"(())"表示第一个选手赢了两场比赛后成为了冠军），那么所有可能的比赛结果数量就是第 n 个卡特兰数。
